What is a linked list?

  • It is a classic abstract data structure
  • It is a list of elements 'pointing' to their next respect next element
  • Not stored in contigous memory
  • Can be fast, O(1) or O(n)
  • Good for Queue or Stack
  • Singly Linked Lists only 'point' to the next element
  • Doubly Linked Lists point to next and previous element

Think of Linked Lists as a bunch of simple rooms connected by a single doors. Each room leads to then next, like how each node points to the next.

The head node points to the first node in the list and is the entry point of the list.

Getting Started with the Node Class

public class Node  {
    //Two class level attributes
    T data; //Generic data of any type
    Node next; //reference to another node of same class

    //Constructor
    //When called the data will be set for this node and next will be set to null as it doesn't exist yet
    public Node(T data){
        this.data = data;
        this.next = null;
    }

    public Node(T data, Node next) {
        this.data = data;
        this.next = next;
    }

}

public void addEnd(T data){
        //First check if head is null
        if(head == null){
            head = new Node(data);
        } else{
            Node iterator = head;

            while(iterator.next != null){ //iterate over linked list until you encounter a null value, at that point you are at the end. It is ok if the next become null, that means currently we are on the end.
                iterator = iterator.next;
            }

            //Upon getting at the end
            iterator.next = new Node(data);
        }
    }

Implementing remove

Getting Started with the Node Class

 public void remove(T data){
        if(head == null){
            return;
        }

        if(head.data == data){
            popFront();
        } else{
            Node iterator = head;
            Node previous = null;

            while(iterator.next != data && iterator.next != null){ //iterator until the next node is the data or no data is found and we get to the end
                previous = iterator; //keep track of the iterator
                iterator = iterator.next;
            }

            //Once finished with whlie we have either reached end or reached data we wana delete
            if(iterator.next == null && iterator.data != data){
                return; //Data is not found, does not exist
            } else  if (iterator.next == null && iterator.data == data){
                previous.next = null;
            }
            else{
                previous.next = iterator.next;//We found the data. Now skip over the found data and point to the data after it, thus removing it from the list
            }
        }
    }

Getting Started with the Node Class

public int size(){
        int count = 0;

        if (head == null){
            return count;
        }

        Node iterator = head;
        count++;
        while(iterator.next != null){
            iterator = iterator.next;
            count++;
        }

        return count;

    }

Implementing Contains

Getting Started with the Node Class

public boolean contains(T data){
        if (head == null){
            return false;
        }

        Node iterator = head;

        if(iterator.data == data){
            return true;
        }
        while(iterator.next != null){
            iterator = iterator.next;

            if(iterator.data == data){
                return true;
            }
        }
        return false;
    }
