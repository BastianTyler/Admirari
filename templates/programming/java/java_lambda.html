Lambda's in Java

What is a lambda expression?

Lambda expressions are handy for simplying code by removing boilerplate and making functional interfaces easier to use. They provide a means to create methods without having to create a class.

They act as a concise way to represent an anonymouse function(a function without a name).

Vocab check: What is a functional interface?

Answer: An interface with a single abstract method. Examples are Runnable, Comparator, etc.

Lambda expressions have three parts:

  • The Parameter List: List of input parameters enclosed in parentheses. Methods with no parameters use an empty set of parentheses
  • The Arrow Operator: Seperates the parameter list from the body
  • The Body: Implementation of the lamda expression. Can be a single expression or a block of code surrounded by curly braces.
