What is Cramers Rule?

Cramers Rule is a technqiue for finding the solutions of a system of linear equations using the power of the determinant.

The rule states:

If \(det(A)\neq 0\), then the unqie solution to \(A\) is replaced by \(b\) is \(x = (x_1,x_2,...x_n)\), where

$$ x_k = det(B_k)/det() $$

Finding the inverse with the identity matrix

So how do we find the inverse? Well, if there is one, we can make an augmented matrix of the matrix we want the inverse of and of the identity.

We start with matrix A

Our goal is to transform Matrix A into the identity matrix. Each row operation we perform on A should occur on the identiy. We augment A to make this obvious.

And the resulting matrix should be A inverse! We can check this by multiplying the original A by the inverse of A.
